# Miniguia de Estudos: Lógica de Programação e Estruturas de Dados com NotebookLM

## Contexto e Objetivos
Este repositório contém o resultado do meu projeto prático desenvolvido para a **DIO**, utilizando o **NotebookLM** (IA do Google) como ferramenta de aprendizagem ativa. 

O objetivo central deste caderno temático é aprofundar o entendimento sobre **Lógica de Programação e Estruturas de Dados**, servindo como material de apoio para minha preparação para o programas de estágio/trainee e concursos na área de tecnologia.

**Objetivos específicos:**
* Dominar conceitos de algoritmos e complexidade.
* Organizar uma base de conhecimento consultável para revisões rápidas.
* Praticar engenharia de prompts voltada para educação técnica.

## 📚Fontes
Para alimentar a inteligência do NotebookLM, selecionei as seguintes fontes:

* **Algoritmos em linguagem C - Paulo Feofiloff** (https://www.ime.usp.br/~pf/algoritmos-livro/downloads/Algoritmos-em-linguagem-C.pdf)
* **Documentação MDN - JavaScript Guide** (https://developer.mozilla.org/pt-BR/docs/Web/JavaScript/Guide)
* **Introdução aos Algoritmos - Afonso Paiva** (https://sites.icmc.usp.br/frasson/sme305/slides/algortimos.pdf)
* **Artigo: Pensamento Computacional e Resolução de Problemas - Helena Campos e Diana Leite** (https://parc.ipp.pt/index.php/elearning/article/view/6377/3636)
